# R.O.B COMMAND CENTER - БЭКАП

Данная папка является резервной директорией для старой версии приложения (Vanilla JS/HTML).

**Внимание:** 
Чтобы обновить ваш внешний репозиторий GitHub (\`training_jump\`), вам нужно воспользоваться функцией **"Export to GitHub"** в настройках AI Studio (кнопка в правом верхнем углу интерфейса).
AI Studio загрузит эту структуру вместе с обновленным приложением React/Vite прямо в ваш GitHub.
